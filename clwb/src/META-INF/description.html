Bazel build system integration for CLion, enabling seamless C++ and C development with fast, scalable builds.


Main features:

  • Import and sync Bazel projects with CLion's project model
  • Build, run, test, and debug Bazel targets for C++ and C (cc_binary, cc_test, cc_library)
  • Smart editing support for BUILD files and Starlark code (completion, navigation, refactoring)
  • Configure projects with .bazelproject files to define scope and build settings
  • Run configurations with gutter icons for quick executable targets and tests
