Bazel for CLion Plugin

Import Bazel project and develop C++ and C code with fast and accurate IDE code insight. Build, run, test, and debug cc_binary, cc_test, cc_library targets right from CLion on all major platforms. First-class Windows support including the MSVC toolchain.

Main features:

  • Bazel sync: one click sync of CLion's project model with your Bazel project
  • Run, Debug, Test Bazel targets for C and C++: cc_binary, cc_test, cc_library
  • Test Framework Support: run and debug cc_test targets with integrated support for GoogleTest and Catch2
  • Cross-platform: develop and debug C and C++ Bazel projects in CLion on all major platforms, including Windows, macOS and Linux
  • Project View (.bazelproject): define workspace scope and Bazel flags (faster sync and indexing for large monorepos)
  • BUILD file and Starlark smart editing (BUILD, .bzl): completion, navigation, and refactor-friendly code insight
